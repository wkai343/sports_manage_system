# 数据库课设

## 项目简介

本项目是一个使用`SQLite`数据库进行设计的`C++`课程设计项目，实现了一个简单的运动会管理系统。

## 项目结构

1. tdialoglogin: 登录界面  
    窗口移动代码使用王维波老师Qt教程的代码
2. tcomboboxdelegate: 自定义下拉框  
    用于在表格中显示下拉框, 复制自王维波老师Qt教程
3. adminwindow: 管理员界面  
    管理员可以对运动员、项目、裁判员等信息进行增删改
4. athwindow: 运动员界面
    运动员可以查看自己的信息
5. judgewindow: 裁判员界面
    裁判员对运动员的进行打分
